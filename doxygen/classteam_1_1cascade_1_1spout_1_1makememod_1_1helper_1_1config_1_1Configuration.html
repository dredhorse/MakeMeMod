Detailed Description

Here you will find all the information you will normally want to change to your liking. If you make changes to the CONFIG enums increase the configCurrent number and the plugin will handle autoupdating if configured.

The following is the default configuration of the plugin if you don't change it via the corresponding setters or the enums TILL the config is loaded:

  • configFile = config.yml
  • debugLogEnabled = false
  • debugLogToFile = true
  • checkForUpdate = true
  • configAutoUpdate = true
  • configAutoSave = true
  • configLogEnabled = true
  • disableOnOlderBuilds = false
