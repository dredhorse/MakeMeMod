Detailed Description

Logger is a HelperClass which allows you to use the full logging levels instead of just INFO, WARNING and SEVERE. It is using a workaround to accomplish this by logging the not existing levels via a custom tag and the INFO level. It also adds a DEBUG log function which will even printout the module.method and line which called that function if logging is done via FINER and higher.
It allows you to log to a file INSTEAD of logging to a console or to log to both. If you enable Logging to FILE everything starting from CONFIG onwards will be logged to file only, including DEBUG.
On Top of that it supplies a enableMessage and disableMessage method which will log via INFO level a message.
